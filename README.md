## Customer Management System - Frontend

## 📌 Project Description
This is the frontend of the Customer Management System developed using React.  
It provides a user-friendly interface to manage customers, including their mobile numbers, addresses, family members, and bulk upload functionality.

---

## 🚀 Technologies Used
- React.js
- Bootstrap
- Axios
